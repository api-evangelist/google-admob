# Google AdMob API

The Google AdMob API provides programmatic access to information about AdMob accounts, allowing publishers to retrieve details about ad sources, ad units, apps, and mediation groups, and to generate campaign, mediation, and network reports for mobile app monetization.

## APIs

- **Google AdMob API** - Programmatic management of mobile app monetization including ad units, apps, mediation groups, and reporting.

## Artifacts

- [APIs.yml](apis.yml) - API discovery document following the APIs.json specification.
- [OpenAPI](openapi/openapi.yml) - OpenAPI 3.1.0 specification for the AdMob API.
- [JSON Schema](json-schema/json-schema.yml) - JSON Schema (draft 2020-12) for API resource models.
- [JSON-LD Context](json-ld/json-ld.yml) - JSON-LD context mapping for linked data interoperability.
